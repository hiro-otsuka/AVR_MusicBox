＝＝＝＝ EEPROM 書き込みツールの使用方法 ＝＝＝＝
■ハードウェアの用意
 Arduino と EEPROM をI2Cで接続する。
 Arduino IDE を起動し、tools\ ディレクトリに格納された eepromwriter.ino スケッチを Arduino に書き込む。

■ツールの使用
 TeraTerm等で、ツールスケッチを書き込んだ Arduino に接続する。
 ※Arduino IDE のシリアルモニタでは複数行のペースト等が行えないので、TeraTermを使用する。
 以下のコマンドで HEX ファイルの書き込みを行う。
  h ... コマンドリストを表示（TeraTerm接続直後も本コマンドで画面表示可能）
  f ... 利用可能な I2C アドレスを表示
  i ... I2C アドレスを変更（初期値は 0x50）
  s ... EEPROM のR/Wアドレスを設定（初期値は 0x0000）
  c ... R/Wアドレス上の 1バイト分を書き換え
        （R/Wアドレスは +1 されるので連続実行が可能）
  w ... HEXファイルのアドレス（通常は0x0000から）に従ってファイルを書き込み
        （書き込み完了時点のアドレスが R/Wアドレスに記録される）
  a ... R/Wアドレス以降にHEXファイルの内容を書き込み
        （wコマンドや aコマンド後の追加書き込みに使用）
  r ... 0x0000 から 1024バイト分読み込んで表示する
        （読み込み終了時点のアドレスが R/Wアドレスに記録される）
  n ... R/Wアドレスから 1024バイト分読み込んで表示する
        （rコマンドや nコマンド後の追加読み込みに使用）
 c,w,a,r,n のコマンドは、実行後に R/Wアドレスが更新されるため、書き込みと読み込みの操作を交互に実施する場合はR/Wアドレスの状態に注意が必要。

■書き込み方法
 w または a コマンドを実行後、HEX ファイルの中身をコピー＆ペーストすることで書き込まれる。
 TeraTerm であれば複数行を一度にペースト可能。
